# What is MAXIMM?
A new format for immersive cinema. MAXIMM stands for Maximal Immersion. It is a way of presenting films/shorts via the VR180 viewing format, along with a certain set of rules that ensure optimal experience for your audience. Keep in mind this is an open source format, meaning anyone can contribute to it, and improvise on it, no legal barriers needed. 
# What is this repository about?
This repository works as a starter guide to present your creative vision in the MAXIMM format. It contains instructions, a simple workflow suggestion, and a ready-to-animate setup in Blender that has all the minimum specs configured for you, so you can start making your Maximally Immersive films right away!
